# Flowstate
The performant, narrow text editor for debaters.

# Building
To build Flowstate, you need to have [Cargo](https://doc.rust-lang.org/cargo/) installed. You also need to enable nightly rust; we use it for compiler optimizations. You can do this by running `rustup override set nightly`.
Run `cargo run --package flowstate --release` to build and run.
